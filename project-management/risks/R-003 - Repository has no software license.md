# R-003 — Repository has no software license

## Risk

The public repository is intentionally unlicensed, so external reuse and
contribution expectations are unresolved.

## Mitigation

- [ ] #task Owner selects a repository license before accepting external reuse or contributions

Do not add a license or imply contribution rights without the owner's explicit
choice.
